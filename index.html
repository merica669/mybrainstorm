üåü What is MyBrainstorm.ai?

MyBrainstorm.ai is a comprehensive web application designed to be an AI-powered academic partner for students. It aims to help users overcome common academic hurdles by assisting with brainstorming, essay structuring, and understanding complex topics through a suite of personalized tools, including a prompt generator and an interactive AI tutor.

üìö The Problem It Solves

‚ñ∂

The Challenge

Students at all levels often face significant challenges, including:

  • Writer's Block: Difficulty starting an assignment or generating initial ideas.
  • Structuring Arguments: Uncertainty on how to organize thoughts into a coherent essay or report.
  • Grasping Complex Subjects: Needing clarification on difficult topics in a way that is tailored to their learning style.
  • Lack of Peer Feedback: Working in isolation without a community to share ideas and receive constructive criticism.

The MyBrainstorm.ai Solution

The application tackles these issues by providing a personalized and multi-faceted learning environment that:

  1. Captures Student Context: A detailed onboarding process gathers the user's academic level, field of study, and learning style.
  2. Generates Tailored Prompts: Creates personalized, Socratic-style prompts that guide students' thinking rather than just giving answers.
  3. Offers an Interactive AI Tutor (Pro): Provides an integrated chat experience for in-depth exploration, guided by an AI persona that acts as an academic coach.
  4. Fosters Community Learning: Includes a "Study Hall" for peer-to-peer feedback and a "Prompt University" for inspiration.
üí° Key Insight: The application's strength lies in its focus on process over product. It aims to teach students *how* to think and structure their work, using AI as a Socratic guide rather than a simple answer machine.

üíª Technology Stack

‚ñ∂

Frontend Technologies

  • HTML5 & CSS3: Standard markup and styling.
  • Tailwind CSS: A utility-first CSS framework for rapid, modern UI development.
  • Vanilla JavaScript: All client-side logic, state management, and API calls are handled with plain JavaScript.
  • Local Storage: Used to persist the user's profile and Pro version status across browser sessions.

AI Integration (Google Gemini)

  • Model for Prompt Curation & Chat: `gemini-2.5-flash-preview-05-20` is used to both engineer the initial curated prompt and power the interactive AI Tutor chat.
  • Model for Text-to-Speech: `gemini-2.5-flash-preview-tts` is used for the audio summary feature in the Pro version.

üöÄ Suggestions for Improvement

üõ°Ô∏è 1. Critical: Secure the API Key

‚ñ∂

The most urgent issue is the exposed API key. This must be fixed by creating a backend proxy server.

Recommended Solution:

  1. Create a simple server-side application (e.g., using Node.js/Express, Python/Flask, or a serverless function like Google Cloud Functions).
  2. Store the Gemini API key securely as an environment variable on the server.
  3. Create an API endpoint on your server (e.g., `/api/generate`).
  4. The frontend sends the user's prompt and chat history to your server endpoint. Your server then securely calls the Gemini API and relays the response back to the client.
// Example Node.js/Express backend endpoint app.post('/api/generate', async (req, res) => { const { systemInstruction, contents } = req.body; const GEMINI_API_KEY = process.env.GEMINI_API_KEY; // Securely loaded // Logic to call the actual Gemini API from the server const response = await callGoogleAPI(GEMINI_API_KEY, { systemInstruction, contents }); res.json(response); });

üíæ 2. Enhance Data Persistence

‚ñ∂

Currently, `localStorage` ties the student's profile and chat history to a single browser. A real user would expect to access their profile and chats from any device.

Recommended Solution:

  • Implement User Authentication: Add a simple sign-in system (e.g., using Firebase Authentication or Auth0) to create user accounts.
  • Use a Database: Store user profiles, chat histories, and Study Hall posts in a cloud database like Firestore. This allows users to log in anywhere and retrieve their data.
  • Benefit: This foundational step enables future features like "Saved Chats," "My Assignment History," and a more robust community forum.

‚ú® 3. Expand Academic Features

‚ñ∂

Suggestions for New Features:

  • Citation Generator: After a brainstorming session, offer a tool to help format sources and citations in various academic styles (APA, MLA, etc.).
  • Argument Mapper: A visual tool where the AI helps the user map out their thesis, main points, and supporting evidence in a flowchart format.
  • Interactive Study Hall: Enhance the forum with threaded replies, upvoting, and the ability to tag posts by subject, making it a more dynamic community hub.
  • Source Analysis Tool: Allow users to paste text from a source document and have the AI generate a summary, identify the main arguments, or suggest counter-arguments.
